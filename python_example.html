Static Configuration¶

This example will be run against the baseline input file. The water depth is \(350\) meters. The space preceeding the repeat 120 240 solver option flag is removed to enable duplication of the mooring geometry twice with \(120^\circ\) and \(240^\circ\) offsets about the \(Z\) axis.
